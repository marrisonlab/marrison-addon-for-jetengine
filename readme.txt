Adds advanced custom grid layouts and listing titles to JetEngine Listing Grid widget in Elementor.

== Description ==

Marrison Addon for JetEngine extends the functionality of the JetEngine Listing Grid widget.

**Features:**

*   **Custom Grid Layouts:** Break free from standard columns with complex, asymmetrical layouts (e.g., 1 Left - 4 Right).
    *   Responsive controls for Desktop, Tablet, and Mobile.
    *   Applies layout to the first batch of items.
*   **Custom Listing Title:** Add a custom title above your Listing Grid with full styling options.
    *   Dynamic Tags support.
    *   Responsive alignment, margin, and padding.
    *   Typography and color controls.

**Available Layouts:**
*   **1 Left - 4 Right**: Displays one large item on the left and four smaller items on the right. (5 items)
*   **4 Left - 1 Right**: Displays four smaller items on the left and one large item on the right. (5 items)
*   **1 Left - 2 Right**: Displays one large item on the left and two smaller items on the right. (3 items)
*   **2 Left - 1 Right**: Displays two smaller items on the left and one large item on the right. (3 items)

== Installation ==

1.  Upload the plugin folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Open a page with Elementor and edit a **Listing Grid** widget.
4.  Go to the **Content** tab and look for the **Marrison Addon Settings** section.
